[![Kampyle Integrator for WordPress](wp-content/uploads/2010/04/kampyle-integrator.png "Kampyle Integrator")](wp-content/uploads/2010/04/kampyle-integrator.png)We've just converted out Joomla version of the Kampyle Integrator in to a nice little neat WordPress version. You can now easily integrate the Kampyle feedback analytics form system on to any WordPress website without the need of modifying your website's templates or widgets.

This plug-in will help with the implementation of the Kampyle css and javascript code needed to get Kampyle running on your WordPress site.

You will need to sign up to [Kampyle](https://www.kampyle.com/feedback-form-kampyle/?r=wordpress) if you don't already have an account. – [Kampyle Signup](https://www.kampyle.com/feedback-form-kampyle/?r=wordpress)

Install and activate the plugin, go to the settings for the plugin and paste in your embed code from Kampyle.

Easy as that.

Please note that you do need to have a Kampyle account in order to get the code required for the integration.  
[https://www.kampyle.com/feedback-form-kampyle/?r=wordpress](https://www.kampyle.com/feedback-form-kampyle/?r=wordpress "link to Kampyle Sign up for WordPress")
